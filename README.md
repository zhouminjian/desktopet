# 🐱 DesktopPet - 桌面宠物 Pii

> 一只生活在你的屏幕上的 AI 小猫伙伴，让电脑不再冰冷，让陪伴充满惊喜。

---

## ✨ 这是什么？

**DesktopPet** 是一款轻量级桌面宠物应用。可以自定义小猫名字和性格，小猫会常驻在你的桌面上，它会发呆、卖萌、打盹，还会感知你的操作节奏——当你忙碌打字时给你加油，当你长时间不理它时偷偷睡着。

它不只是一个桌面图标，而是一个有性格、会成长、懂你的**数字伙伴**。

## 💡 核心价值

### 🧡 情感陪伴
电脑是冰冷的机器，但 Pii 不是。它会主动找你、回应你、依赖你——让你的屏幕有了温度。

### 🎭 性格成长
每只 Pii 都是独一无二的。你选择它的性格，你的互动方式塑造它的行为。活泼、温柔、幽默、傲娇——你的 Pii，你定义。

### 🌱 养成乐趣
喂食、抚摸、陪伴——Pii 会从幼年成长为少年、青年。等级提升、经验积累，每一步成长都有你在。

### 🔍 智能感知
Pii 不仅仅被动等你点击。它能感知你的键盘节奏，在你高强度工作时送来鼓励，在你离开时安静入睡。

## 🚀 快速开始

### 环境要求
- Python 3.10+
- Windows 系统

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动
```bash
python main.py
```

首次启动会弹出性格选择和宠物命名窗口，之后 Pii 就会出现在你的桌面右下角。

---

## 🏗️ 技术架构

- **UI 框架**：PySide6（透明置顶窗口、序列帧动画）
- **核心引擎**：纯 Python 状态机 + 数值引擎
- **感知系统**：pynput 键盘监听
- **数据持久化**：JSON 存档（`%APPDATA%\DesktopPet\`）

采用 MVC 架构，逻辑与界面分离，便于后续扩展。

```
├── main.py                # 程序入口
└── requirements.txt       # 依赖列表
```
