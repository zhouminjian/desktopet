# 《桌面宠物》产品设计全案

## 1. 产品定位 & 核心愿景

**产品名称**：桌面宠物（暂定）

**一句话核心定义**：
> 一只生活在你的屏幕上的AI小猫伙伴，让电脑不再冰冷，让陪伴充满惊喜。

**产品愿景**：
打造新时代桌面宠物标杆，以轻AI驱动情感陪伴、拟物交互与可持续养成体系，成为用户电脑生活中的“数字宠物灵魂”。

**核心卖点（3 大创新点）**
1. **真实桌面交互**：可在窗口边缘爬行、趴在任务栏、被鼠标逗玩
2. **小猫独特性格成长**：行为随陪伴方式变化，形成不同性格标签
3. **感知系统进化**：理解用户操作，主动求关注、送惊喜

**主用户群画像**
| 用户类型 | 动机 | 使用场景 |
|---------|-----|---------|
| 学生党、白领 | 获得陪伴、减压 | 工作/学习期间 |
| 二次元 & 宠物爱好者 | 收藏、装饰桌面 | 个性展示 |
| 轻度游戏玩家 | 养成进阶、任务奖励 | 闲暇娱乐 |

**市场机会点**
- 情绪陪伴需求显著增长
- 桌面生态回潮（直播、内容工作者久坐场景多）
- 怀旧情绪驱动高热度（QQ宠物经典IP认知）

---

## 2. 核心体验循环

> **陪伴 → 互动 → 成长 → 情感连接**

```
陪伴：常驻桌面存在感
  ↓
互动：喂食、抚摸、小游戏
  ↓
成长：获取经验，解锁行为与装扮
  ↓
情感连接：主动找你、表达依恋
  ↺（循环提升依赖感）
```

## 4. 桌面交互系统设计（Q1: A+B方案）

**表现能力**：
- 走：随机巡逻屏幕边缘与窗口间
- 爬：沿窗口边框移动、悬停
- 坠：误步掉落时有摔倒行为（物理动画）
- 依附：趴任务栏、靠着窗口睡觉
- 触发型动作：点击头会瞪你、摸肚子会眯眼

**用户影响事件**（感知系统已实现基础版）：
| 用户操作 | 宠物反应 | 实现状态 |
|--------|---------|---------|
| 连续敲键盘很久（>100次/分钟） | 送来鼓励气泡”要加油！我就在这里！” | 已实现 |
| 长时间无操作（≥120秒） | 自动进入睡眠 | 已实现 |
| 精力不足（≤20） | 触发 tired 动画，气泡提醒 | 已实现 |
| 心情低落（≤35） | 躺下（lie），邀玩气泡 | 已实现 |
| 鼠标不动（发呆） | 小跑来撒娇求关注 | 待开发 |
| 关闭大窗体 | 摔倒后埋怨你（表情+文字） | 待开发 |

## 5. IP角色设定：小猫性格 & 成长机制

**主角小猫** - “Pii”

### 成长阶段
| 阶段 | 外观变化 | 解锁能力 | 性格成长 |
|---|---|---|---|
| 幼年 | 圆滚滚、小短腿 | 基础互动 | 依赖、撒娇多 |
| 少年 | 身形拉高、装备变化 | 外出打工 | 好奇、偶尔倔强 |
| 青年 | 姿态成熟、动作丰富 | 协助提醒用户 | 责任感与陪伴感强 |

### 性格选择机制
- 创建小猫时，用户从预设性格标签中选择3个偏好风格
- 系统会自动禁止选择对立性格，确保性格一致性
- 性格标签会影响：
  - 行为概率（撒娇、调皮、安慰）
  - 对话语气（活泼、温柔、幽默）
  - 成长阶段动作与表情差异

### 情绪树与行为权重
- 根节点：心情值
- 节点：快乐、委屈、兴奋、疲倦、无聊、想陪你
- 行为权重受性格标签和用户互动影响

**行为示例**
| 用户操作 | 小猫反馈 | 实现状态 |
|--------|---------|---------|
| 连续敲键盘20分钟 | *”要加油！我就在这里！”* | 已实现 |
| 停止操作120秒 | 自动进入睡眠 | 已实现 |
| 精力不足 | *有点累了...* → tired 动画 → 强制睡眠 | 已实现 |
| 心情低落 | 躺下（lie），*主人...陪我玩一会嘛~* | 已实现 |
| 饱食度低 | *好饿...快喂我吃的！* → 饥饿动画 | 已实现 |
| 睡眠中点击 | *太累了...让我再睡会儿* / *好饿...没力气起来* | 已实现 |
| 频繁戳它 | 生气 + 躲图标后面 | 待开发 |
| 长时间不理它 | 偷偷睡觉，回来时扑过来 | 待开发 |

### 智能互动气泡

点击小猫时的反馈不再仅是固定的性格文案，而是根据 **时间 + 宠物状态 + 成长** 多维度智能选择。

**核心机制：条件优先 + 50%概率 + 性格兜底**

```
点击 → 按优先级检查特殊条件
  ├─ 命中条件（50%概率）→ 使用该条件的性格文案池
  └─ 未命中 / 50%降级 → 使用通用 petted 文案池
```

| 优先级 | 维度 | 条件 | 示例文案 |
|--------|------|------|---------|
| 1 | 时间-深夜 | 22:00-5:00 | "这么晚了还不睡？{name}陪你熬夜~" |
| 2 | 状态-饥饿 | hunger ≤ 30 | "摸我不如喂我...肚子好饿" |
| 3 | 状态-精力低 | energy ≤ 40 | "有点累了...但被你摸还是很开心" |
| 4 | 状态-心情高 | happiness ≥ 80 | "今天超开心！因为有你陪着~" |
| 5 | 时间-早晨 | 6:00-9:00 | "早上好！今天也一起加油吧~" |
| 6 | 时间-午后 | 12:00-14:00 | "午饭吃了吗？别忘了吃饭哦~" |
| 7 | 成长-里程碑 | age_days 整10天 | "我们已经在一起{days}天了呢~" |
| 8 | 通用 | 无条件 | 现有 petted 文案（性格相关） |

每种性格（活泼/温柔/懒散/好奇）对应 7 个条件 × 3-4 条文案，共约 100 条新增文案，保证互动体验丰富不重复。
